# 概要

- レモンを塩漬けにしたもの
- 塩味、酸味、風味付けに便利

# 原材料

- 塩
- レモン

# 用途

- 肉類の下ごしらえ(マリネ液の代わり)
- ロースト肉用のソース
   - 肉の焼き汁と塩レモン、バター、刻んだハーブを混ぜ、中火のフライパンでなじませる
- サラダのドレッシング
   - 塩味と酸味をこれでまかない、ビネガーや塩はその分抑えめに
